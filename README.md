# Msc-projects
Superstore Sales Data Analysis

This project analyses a retail Superstore dataset using Python to identify key business insights related to sales performance and profitability.

Tools Used
	•	Python
	•	Pandas
	•	Matplotlib
	•	Jupyter Notebook

Dataset

The dataset contains sales records including order information, customer details, product categories, and profit metrics.

Total rows: 9994
Total columns: 21

Analysis Performed

Sales by Category

Analyzed total sales across product categories to understand which category generates the highest revenue.

Profit by Region

Compared profit across regions to identify the most profitable markets.

Profit by Sub-Category

Identified which products generate profit and which products cause losses.

Monthly Sales Trend

Analyzed sales trends over time to identify growth patterns and seasonal fluctuations.

Key Insights

• Technology products generate the highest sales
• The West region contributes the highest profit
• Tables and Bookcases show negative profit
• Sales show an overall upward trend from 2014–2017
